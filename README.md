# retail-data-cleaning-analysis
Data cleaning, preparation, and exploratory analysis on retail transaction data using Python &amp; Pandas.
## 🛒 Retail Data Cleaning & Preparation

This project focuses on cleaning and preparing real-world retail transaction data
for analysis. It covers the full data wrangling pipeline — from raw CSV files to
a clean, analysis-ready dataset.

### 📂 Dataset
- `Retail_Data_Transactions.csv` — Customer transaction records (customer_id, date, amount)
- `Retail_Data_Response.csv` — Customer response/campaign data

### 🔧 What's covered
- Merging multiple datasets on a common key (`customer_id`)
- Exploring data types, shape, and summary statistics
- Handling missing values
- Type conversion (dates, integers)
- Outlier detection using Z-Score method
- Feature engineering (extracting month from date)
- Business insights:
  - Top 3 months by total transaction amount
  - Top 5 customers by number of orders

### 🛠️ Tools & Libraries
- Python, Pandas, NumPy
- SciPy (Z-score outlier detection)
- Seaborn & Matplotlib (visualization)

### 📊 Key Takeaways
Hands-on practice with real retail data to build a clean foundation
for further analysis or machine learning pipelines.
